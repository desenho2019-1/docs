1. Indrotução.

4. Product Backlog

## 1 Introdução
<p>Este documento tem como foco explicitar os requisitos para aplicação translate.me no contexto da matéria de desenho e arquitetura de software.</p>

## 2. Objetivo
<p>O obejtivo principal desse documento é deixar de uma forma mais clara, com hístorias de usuários, em uma tabela os requisitos retirados pelo grupo para o app em questão.</p>

## 3. Metodologia
