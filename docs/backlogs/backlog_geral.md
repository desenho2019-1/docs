# Backlog


## Introdução  

Este Backlog foi elaborado de acordo com os backlogs e introspecções individuais realizados pelos integrantes do grupo. Aqui consta todos os requisito elicitados e filtrados nas técnicas citadas, assim é elaborado todo o quadro de historia de usuário do projeto

## Objetivo

A finalidade deste documento é definir e priorizar as tarefas a serem feitas durante o projeto.
